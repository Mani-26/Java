Introduction to Java

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

It is one of the most popular programming languages for building applications ranging from mobile to enterprise systems.

Features of Java

  • Platform Independent: Write Once, Run Anywhere (WORA) due to Java's bytecode.
  • Object-Oriented: Everything in Java revolves around objects and classes.
  • Robust: Java provides strong memory management and exception handling.
  • Secure: Java is designed with security in mind, making it a preferred choice for web development.
  • Multithreading: Supports multiple threads of execution, making applications faster and more efficient.

Setting Up Java

To start programming in Java, follow these steps:

  1. Download the latest JDK from the Oracle website.
  2. Install the JDK and set up the `JAVA_HOME` environment variable.
  3. Verify the installation by running the following command in your terminal or command prompt:
    4. java -version
  4. Choose an IDE like IntelliJ IDEA, Eclipse, or Visual Studio Code to write and run your code.

Basic Syntax of Java

Here’s a simple Java program:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Explanation:

  • public class: Defines a class named `HelloWorld`.
  • main method: The entry point of a Java program.
  • System.out.println: Prints a message to the console.

Control Structures in Java

Control structures help you control the flow of execution. Here's an overview:

Conditionals

Example of an `if-else` statement:

int number = 10;
if (number > 0) {
    System.out.println("Positive number");
} else {
    System.out.println("Negative number");
}

Loops

Example of a `for` loop:

for (int i = 0; i < 5; i++) {
    System.out.println("Count: " + i);
}

Introduction to Object-Oriented Programming (OOP)

Java is an object-oriented language, which means it is based on objects and classes. Here are some OOP concepts:

  • Class: A blueprint for objects.
  • Object: An instance of a class.
  • Inheritance: Allows one class to inherit fields and methods from another.
  • Encapsulation: Hides internal details and shows only functionality.
  • Polymorphism: Allows methods to perform different tasks based on the object.

Example of a class and object:

class Animal {
    String name;

    void makeSound() {
        System.out.println("Animal makes sound");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal dog = new Animal();
        dog.name = "Dog";
        dog.makeSound();
    }
}
